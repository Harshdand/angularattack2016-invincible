About Smart Todo X

Smart Todo application is based on Eisenhower Matrix

Important/Urgent quadrant are done immediately and personally e.g. crises, deadlines, problems.

Less Important/Urgent quadrant are delegated e.g. interruptions, meetings, activities.

Important/Less Urgent quadrant get an end date and are done personally e.g. relationships, planning, recreation.

Less Important/Not Urgent quadrant are dropped e.g. time wasters, pleasant activities, trivia.

Using the Eisenhower Decision Principle, tasks are evaluated using the criteria important/less important and urgent/less urgent, and then placed in according quadrants in an Eisenhower Matrix.
